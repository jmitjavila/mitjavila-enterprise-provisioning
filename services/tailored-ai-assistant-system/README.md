# Tailored AI Assistant System

**Status:** Draft

This folder is the home for the **Tailored AI Assistant System** service.
Use it to define how we design, build, and maintain custom AI assistants for clients.

It should eventually cover:
- Service definition (what problems we solve with AI assistants)
- Platforms / stack we use (OpenClaw, LLM providers, integrations)
- Design process (requirements, persona, workflows, safety constraints)
- Deployment & monitoring playbooks
- A `clients/` folder for per-client configurations and automations

Start by describing the offer in `SERVICE_SCHEMA.md` (tiers, capabilities, boundaries, and support model).
